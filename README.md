Airflow ETL Pipeline Project

This is an automated ETL pipeline built using Apache Airflow, Python, and PostgreSQL.

It performs:
- Data extraction
- Data transformation
- Data loading into database
- Workflow scheduling using Airflow DAGs

Tech Stack:
- Apache Airflow
- Python
- PostgreSQL
